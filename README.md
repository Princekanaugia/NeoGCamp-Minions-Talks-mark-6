# NeoGCamp-Minions-Talks-mark-6
**Minions Talks** is a fun translator which convert your english text in Minions language. This App was made as part of **level Zero** of **NeoGCamp**. 
## Technologies used
 - HTML5
 - CSS3
 - JavaScript
 - Netlify for CI/CD
